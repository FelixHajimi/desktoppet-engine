## 常见问题汇总

---

### 启动问题

**Q: 程序启动后没有任何反应？**

**A:** 请检查以下事项：

1. 确认 Python 环境已安装 PySide6：`pip install PySide6`
2. 检查根目录下是否存在 `setting.json` 文件
3. 查看 `last.log` 日志文件，确认是否有错误信息

---

**Q: 启动时提示找不到桌宠？**

**A:** 检查 `setting.json` 中的 `desktopPet` 值是否与 `data/` 目录下的文件夹名称一致。

- 例如 `"desktopPet": "Example"` 对应 `data/Example/`
- 文件夹名称区分大小写

---

**Q: `setting.json` 文件丢失或格式错误怎么办？**

**A:** 程序会报错并退出。可以从项目中复制一份正确的模板，或手动创建：

```json
{
  "desktopPet": "Example",
  "debug": false,
  "language": "zh-cn",
  "dataDir": "data",
  "desktoppetResourceDir": "res",
  "pluginFileName": "enter",
  "pluginObjectEntry": "enter",
  "logPath": "./last.log",
  "imageSize": [128, 128]
}
```

---

**Q: 日志文件在哪里？**

**A:** 日志文件路径由 `setting.json` 中的 `logPath` 指定，默认为根目录下的 `last.log`，包含程序运行时的所有日志输出。开启调试模式后会输出更详细的信息。

---

### 配置问题

**Q: 修改配置后没有生效？**

**A:** 修改 `setting.json` 或 `config.json` 后需要重启程序。

---

**Q: 修改语言后没有变化？**

**A:** 确认 `language` 值是否为支持的语言代码，修改后需要重启程序。

目前支持的语言：
`zh-cn`、`zh-tw`、`en-us`、`ja-jp`、`ko-kr`、`ru-ru`、`fr-fr`

---

### 物理行为问题

**Q: 桌宠卡在屏幕边缘不动了？**

**A:** 检查 `config.json` 中的 `fri` 参数。如果摩擦力设置过大，可能导致速度无法抵消重力。建议 `top` 和 `bottom` 设置在 `1~5` 之间。

---

**Q: 桌宠掉落速度太快或太慢？**

**A:** 调整 `acc` 参数中的 `y` 值：

- 值越大下落越快
- 推荐范围 `0.8 ~ 2`

---

**Q: 桌宠撞击边缘后弹跳异常？**

**A:** 检查 `ela` 参数。值越大反弹越强，例如 `10` 表示反弹 10% 的速度。

| 方向     | 推荐值 |
| -------- | ------ |
| 底部反弹 | `5`    |
| 左右反弹 | `10`   |
| 顶部反弹 | `0`    |

---

**Q: 如何让桌宠水平移动？**

**A:** 调整 `acc` 参数中的 `x` 值，或在插件中修改 `state["motion"][0]`。

---

### 动画问题

**Q: 桌宠不显示动画？**

**A:** 确认以下事项：

1. `res/` 目录下存在 `stand.gif` 和 `drop.gif`
2. 文件为有效的 GIF 格式
3. 文件名称大小写正确（全小写）

---

**Q: 桌宠大小不对？**

**A:** 确认 GIF 尺寸与 `setting.json` 中的 `imageSize` 一致。

---

**Q: 透明背景不生效？**

**A:** 确认 GIF 保存时选择了透明背景选项。

---

### 插件问题

**Q: 插件加载失败？**

**A:** 检查以下事项：

1. `config.json` 中的 `plugin` 列表是否与 `plugin/` 目录下的文件夹名称一致
2. 插件目录中是否包含 `setting.json` 中 `pluginFileName` 指定的文件（默认为 `enter.py`）
3. `enter.py` 中是否正确定义了 `pluginName` 和 `menu`
4. 错误日志会写入 `last.log`

---

**Q: 插件菜单没有显示？**

**A:** 确认：

1. 插件已正确配置在 `plugin` 列表中
2. `pluginName` 已定义
3. `menu` 字典不为空

---

**Q: 插件自动启动不执行？**

**A:** 确认类中设置了 `self._autoStart = True`。

---

### 打包问题

**Q: 如何将程序打包成 exe？**

**A:** 使用 PyInstaller：

```bash
pip install pyinstaller
pyinstaller --onefile --windowed --name "DesktopPet" main.py
```

---

**Q: 打包后桌宠无法运行？**

**A:** 检查打包时是否包含了 `data/` 目录、`setting.json` 和 `languageMap.json` 文件。可能需要使用 `--add-data` 参数：

```bash
pyinstaller --onefile --windowed --add-data "data;data" --add-data "setting.json;." --add-data "languageMap.json;." main.py
```

---

### 其他问题

**Q: 桌宠窗口被其他程序遮挡？**

**A:** 程序默认置顶，如果被遮挡，检查是否有其他程序强制置顶。可以尝试重启程序。

---

**Q: 如何更换桌宠？**

**A:** 修改 `setting.json` 中的 `desktopPet` 值为目标角色文件夹名称，重启程序。

---

**Q: 如何分享我制作的角色包？**

**A:** 将角色文件夹压缩为 `.zip` 文件，其他用户解压到 `data/` 目录即可使用。

---

**Q: 报错信息看不懂怎么办？**

**A:**

1. 查看 `last.log` 日志文件
2. 将错误信息和日志提交 Issue
3. 加入 QQ 群（109956850）寻求帮助
