## 桌宠配置

桌宠配置存储在 `data/[桌宠名]/config.json` 文件中。该文件定义单个桌宠的行为参数。

```json
{
  "name": "Example",
  "version": "1.0.0",
  "author": "YourName",
  "acc": [0, 1.2],
  "fri": {
    "top": 2,
    "bottom": 3,
    "left": 0,
    "right": 0
  },
  "ela": {
    "top": 0,
    "bottom": 5,
    "left": 10,
    "right": 10
  },
  "plugin": ["message"]
}
```

---

### `name`（必须）

桌宠名称，显示在窗口标题栏。

### `version`（必须）

桌宠版本号。

### `author`（必须）

桌宠作者。

### `acc`（必须）

重力加速度，格式为 `[x方向, y方向]`。

| 方向 | 说明               | 推荐值    |
| ---- | ------------------ | --------- |
| `x`  | 水平加速度         | `0`       |
| `y`  | 垂直加速度（重力） | `0.8 ~ 2` |

### `fri`（必须）

摩擦力。当桌宠到达屏幕边缘时，对应方向的摩擦力会逐渐减小速度。

| 参数     | 说明               | 推荐值  |
| -------- | ------------------ | ------- |
| `top`    | 到达顶部时的摩擦力 | `1 ~ 5` |
| `bottom` | 到达底部时的摩擦力 | `1 ~ 5` |
| `left`   | 到达左侧时的摩擦力 | `0`     |
| `right`  | 到达右侧时的摩擦力 | `0`     |

### `ela`（必须）

弹力系数。当桌宠撞击屏幕边缘时，速度会按百分比反弹。

| 参数     | 说明                 | 推荐值         |
| -------- | -------------------- | -------------- |
| `top`    | 到达顶部时的弹力系数 | `0`            |
| `bottom` | 到达底部时的弹力系数 | `5`（即 5%）   |
| `left`   | 到达左侧时的弹力系数 | `10`（即 10%） |
| `right`  | 到达右侧时的弹力系数 | `10`（即 10%） |

### `plugin`（可选）

需要加载的插件列表，每个元素对应 `plugin/` 目录下的文件夹名称。

```json
"plugin": ["message", "reminder"]
```

如果不使用插件，可以省略该字段：

```json
"plugin": []
```

---

### 动画资源

桌宠需要准备以下动画资源且大小为128x128，存放在 `res/` 目录下：

| 文件名      | 说明                 |
| ----------- | -------------------- |
| `stand.gif` | 站立动画（空闲状态） |
| `drop.gif`  | 掉落动画（运动状态） |
| `icon.gif`  | 窗口图标             |

程序会根据桌宠的运动状态自动切换 `stand` 和 `drop` 动画。
