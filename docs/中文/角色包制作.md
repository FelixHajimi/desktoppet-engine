## 角色包制作

本教程面向设计师和普通用户，教你如何从零开始制作一个桌宠角色包，无需编程基础。

---

### 概述

一个角色包就是一个文件夹，包含角色配置文件和动画素材。用户只需将角色包放入 `data/` 目录，并在 `setting.json` 中指定角色名即可使用。

---

### 目录结构

```
data/
└── 你的角色名/            # 角色包根目录（文件夹名称即角色名）
    ├── config.json       # 角色配置文件
    └── res/              # 素材资源目录
        ├── stand.gif     # 站立动画
        ├── drop.gif      # 掉落动画
        └── icon.gif      # 窗口图标
```

> 注：插件目录 `plugin/` 属于插件开发范畴，角色包制作者无需关心。如果角色包需要附带插件，请参阅[插件开发文档](./插件开发.md)。

---

### 第一步：创建角色文件夹

在 `data/` 目录下新建一个文件夹，名称即为角色名。

```
data/
└── MyPet/                # 例如：角色名称为 MyPet
```

**命名规范**：

- 使用英文字母、数字、下划线
- 不建议使用中文、特殊字符或空格
- 示例：`MyPet`、`cat_white`、`robot_v2`

---

### 第二步：准备动画素材

角色需要准备 3 个 GIF 动画文件，存放在 `res/` 目录下：

```
data/MyPet/
└── res/
    ├── stand.gif         # 站立动画（空闲状态）
    ├── drop.gif          # 掉落动画（运动状态）
    └── icon.gif          # 窗口图标
```

#### 素材要求

| 文件        | 说明              | 建议尺寸 | 格式 |
| ----------- | ----------------- | -------- | ---- |
| `stand.gif` | 站立/空闲状态动画 | 128x128  | GIF  |
| `drop.gif`  | 下落/运动状态动画 | 128x128  | GIF  |
| `icon.gif`  | 任务栏/标题栏图标 | 128x128  | GIF  |

> 实际显示尺寸由 `setting.json` 中的 `imageSize` 控制，素材建议保持正方形比例。

---

### 第三步：编写配置文件

在角色根目录下创建 `config.json` 文件：

```
data/MyPet/
├── config.json           # 配置文件
└── res/                  # 素材目录
```

配置文件的具体字段说明请参阅：[桌宠配置](./桌宠配置.md)

#### 基础配置模板

```json
{
  "name": "MyPet",
  "version": "1.0.0",
  "author": "YourName",
  "acc": [0, 1],
  "fri": {
    "top": 2,
    "bottom": 3
  },
  "ela": {
    "bottom": 5,
    "left": 10,
    "right": 10
  },
  "plugin": []
}
```

> 如果角色不需要插件，`plugin` 数组留空即可。

---

### 第四步：测试角色包

1. 将角色包放入 `data/` 目录
2. 修改根目录的 `setting.json`：

```json
{
  "desktopPet": "MyPet",
  "dataDir": "./data",
  "imageSize": [128, 128],
  "language": "zh-cn",
  "debug": false,
  "logPath": "./last.log",
  "logLevel": 2
}
```

3. 运行 `main.py` 或启动打包后的程序
4. 观察角色是否正常显示，物理行为是否符合预期

---

### 第五步：调整物理参数（可选）

如果角色行为不理想，可以调整 `config.json` 中的物理参数。详见 [桌宠配置](./桌宠配置.md)。

| 问题             | 调整方法                                  |
| ---------------- | ----------------------------------------- |
| 下落太快         | 减小 `acc` 的 `y` 值                      |
| 下落太慢         | 增大 `acc` 的 `y` 值                      |
| 撞击边缘反弹太强 | 减小 `ela` 对应方向的值                   |
| 撞击边缘反弹太弱 | 增大 `ela` 对应方向的值                   |
| 卡在边缘不动     | 检查 `fri` 是否大于 `acc`，适当减小 `fri` |
| 水平方向不动     | 增大 `acc` 的 `x` 值                      |

---

### 完整示例

角色包完整目录结构：

```
data/MyPet/
├── config.json
└── res/
    ├── stand.gif
    ├── drop.gif
    └── icon.gif
```

`config.json` 内容：

```json
{
  "name": "MyPet",
  "version": "1.0.0",
  "author": "YourName",
  "acc": [0, 1.2],
  "fri": {
    "top": 2,
    "bottom": 3,
    "left": 0,
    "right": 0
  },
  "ela": {
    "top": 0,
    "bottom": 5,
    "left": 10,
    "right": 10
  },
  "plugin": []
}
```

---

### 分享角色包

角色包就是一个文件夹，你可以：

1. **压缩分享**：将角色文件夹压缩为 `.zip` 或 `.7z` 文件
2. **提交到项目**：如果质量优秀，可以提交 Issue 或 PR 贡献给项目
3. **发布到社区**：在 QQ 群或论坛分享给其他用户

**安装方式**：用户只需将角色包解压到 `data/` 目录，并在 `setting.json` 中修改 `desktopPet` 为角色目录名即可使用。
