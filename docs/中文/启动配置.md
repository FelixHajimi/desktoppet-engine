## 启动配置

启动配置存储在根目录的 `setting.json` 文件中。该文件控制程序启动时的全局行为。

```json
{
  "desktopPet": "Example",
  "debug": true,
  "language": "zh-cn",
  "dataDir": "data",
  "desktoppetResourceDir": "res",
  "pluginFileName": "enter",
  "pluginObjectEnter": "enter",
  "logPath": "./last.log",
  "imageSize": [128, 128]
}
```

---

### `desktopPet`

启动时需要加载的桌宠目录名称。

程序会根据该名称在 `dataDir` 指定的目录下查找对应的桌宠文件夹。例如 `"Example"` 表示加载 `data/Example/` 目录下的桌宠。

---

### `debug`

全局调试模式开关。

| 值      | 说明                                   |
| ------- | -------------------------------------- |
| `true`  | 开启调试模式，输出详细日志，显示碰撞箱 |
| `false` | 关闭调试模式，仅输出必要信息           |

开启后，右键菜单会额外显示"碰撞箱"和"输出参数"选项，方便开发调试。

---

### `language`

程序使用的界面语言。影响日志输出、右键菜单、对话框以及插件界面。

目前支持的语言：

| 语言             | 代码    |
| ---------------- | ------- |
| 中文（中国大陆） | `zh-cn` |
| 中文（台湾地区） | `zh-tw` |
| 英语（美国）     | `en-us` |
| 日语（日本）     | `ja-jp` |
| 韩语（韩国）     | `ko-kr` |
| 俄语（俄罗斯）   | `ru-ru` |
| 法语（法国）     | `fr-fr` |

---

### `dataDir`

桌宠数据目录名称。

程序会在该目录下查找所有桌宠文件夹。

---

### `desktoppetResourceDir`

桌宠资源目录名称。

每个桌宠的动画素材存放在 `[桌宠目录]/[资源目录]/` 下。

---

### `pluginFileName`

插件入口文件名（不含扩展名）。

程序会加载 `[桌宠目录]/plugin/[插件名]/[插件文件名].py`。

---

### `pluginObjectEnter`

插件入口对象名称。

程序会调用插件中名为 `[插件入口对象]` 的方法。

---

### `logPath`

日志文件路径。

程序运行时的所有日志输出将写入该文件。

---

### `imageSize`

桌宠动画尺寸，格式为 `[宽度, 高度]`。

程序会自动将动画缩放到该尺寸显示。
