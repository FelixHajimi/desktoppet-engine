## 启动配置

启动配置存储在根目录的 `setting.json` 文件中。该文件控制程序启动时的全局行为。

```json
{
  "desktopPet": "Example",
  "debug": true,
  "language": "zh-cn"
}
```

### `desktopPet`（必须）

启动时需要加载的桌宠目录名称。

程序会根据该名称在 `data/` 目录下查找对应的桌宠文件夹。例如 `"Example"` 表示加载 `data/Example/` 目录下的桌宠。

### `debug`（必须）

全局调试模式开关。

| 值 | 说明 |
|------|------|
| `true` | 开启调试模式，输出详细日志，显示碰撞箱 |
| `false` | 关闭调试模式，仅输出必要信息 |

开启后，右键菜单会额外显示"碰撞箱"和"输出参数"选项，方便开发调试。

### `language`（必须）

程序使用的界面语言。影响日志输出、右键菜单、对话框以及插件界面。

目前支持的语言：

| 语言 | 代码 |
| --- | --- |
| 中文（中国大陆） | `zh-cn` |
| 中文（台湾地区） | `zh-tw` |
| 英语（美国） | `en-us` |
| 日语（日本） | `ja-jp` |
| 韩语（韩国） | `ko-kr` |
| 俄语（俄罗斯） | `ru-ru` |
| 法语（法国） | `fr-fr` |
