## 插件

---

### 概述

桌宠引擎支持通过插件扩展功能。每个插件是一个独立的文件夹，包含一个 `enter.py` 入口文件。

插件可以添加右键菜单项，菜单项点击后执行对应的逻辑。插件也可以设置启动时自动执行。

---

### 目录结构

```
data/
└── 桌宠目录/
    ├── config.json          # 桌宠配置
    ├── plugin/              # 插件目录
    │   └── [插件名]/         # 单个插件文件夹
    │       └── enter.py     # 插件入口文件（必须）
    └── res/                 # 桌宠资源
```

---

### 启用插件

在桌宠的 `config.json` 中，通过 `plugin` 字段声明需要加载的插件：

```json
{
  "name": "Example",
  "version": "1.0.0",
  "author": "YourName",
  "acc": [0, 1.2],
  "fri": {
    "top": 2,
    "bottom": 3,
    "left": 0,
    "right": 0
  },
  "ela": {
    "top": 0,
    "bottom": 5,
    "left": 10,
    "right": 10
  },
  "plugin": ["message", "reminder"]
}
```

`plugin` 数组中的每个字符串对应 `plugin/` 目录下的一个插件文件夹名称。

---

### 插件入口文件（enter.py）

每个插件必须包含 `enter.py`，并定义以下内容：

#### 1. `pluginName`（必须）

插件的显示名称，会出现在右键菜单中作为子菜单的标题。

```python
pluginName = "弹窗消息"
```

#### 2. `menu` 字典（必须）

定义右键菜单项。结构为嵌套字典，支持以下键值：

| 键            | 值类型                  | 说明                                             |
| ------------- | ----------------------- | ------------------------------------------------ |
| `"_type"`     | `"$"` 或 `"-"` 或 `"/"` | `$`=菜单项，`-`=分隔符，`/`=子菜单               |
| `"_func"`     | `Callable`              | 点击菜单项时直接调用的函数（无参数）             |
| `"_activate"` | `object`                | 包含 `init`、`activate`、`deactivate` 方法的对象 |

**菜单项（`_type: "$"`）**：

```python
# 方式一：使用 _func 直接绑定函数
{
    "退出": {
        "_type": "$",
        "_func": window.close
    }
}

# 方式二：使用 _activate 绑定功能对象
{
    "消息": {
        "_type": "$",
        "_activate": Information()
    }
}
```

**分隔符（`_type: "-"`）**：

```python
{
    "sep_0": {"_type": "-"}
}
```

**子菜单（`_type: "/"`）**：

```python
{
    "工具": {
        "_type": "/",
        "功能A": {
            "_type": "$",
            "_activate": FunctionA()
        },
        "功能B": {
            "_type": "$",
            "_activate": FunctionB()
        }
    }
}
```

---

### 功能对象（\_activate）

当使用 `_activate` 方式定义菜单项时，对象需要实现以下方法：

#### 1. `init` 方法（必须）

插件初始化时调用，用于保存引擎提供的对象引用：

```python
def init(self, image: QtGui.QMovie, mainTimer: QtCore.QTimer, physicsTimer: QtCore.QTimer, window: QtWidgets.QWidget):
    """
    插件初始化方法

    参数:
        image: QMovie 对象，用于控制桌宠动画
        mainTimer: QTimer 对象，主循环定时器（20ms）
        physicsTimer: QTimer 对象，物理循环定时器（20ms）
        window: QWidget 对象，主窗口
    """
    self._image = image
    self._mainTimer = mainTimer
    self._physicsTimer = physicsTimer
    self._window = window
```

#### 2. `activate` 方法（必须）

用户点击菜单项时调用，接收当前桌宠状态：

```python
def activate(self, state: dict):
    """
    菜单项被点击时的回调方法

    参数:
        state: 桌宠状态字典（可读写），包含以下字段：
            - "pause": bool，是否暂停物理
            - "position": list[int, int]，当前 [x, y] 位置
            - "motion": list[int, int]，当前 [vx, vy] 速度
            - "sep_iter": generator，分隔符序号生成器
            - "update": dict[str, Callable]，每帧执行的函数注册表
            - "screenWidth": int，屏幕宽度
            - "screenHeight": int，屏幕高度
    """
    pass
```

#### 3. `deactivate` 方法（可选）

窗口关闭时调用，用于清理资源：

```python
def deactivate(self):
    """插件清理方法，窗口关闭时调用"""
    pass
```

#### 4. `_autoStart` 实例属性（必须）

在 `__init__` 中定义 `_autoStart` 实例属性，控制是否在程序启动时自动执行 `activate` 方法：

```python
class Information:
    def __init__(self):
        self._autoStart = True  # 启动时自动执行
        # 或
        self._autoStart = False  # 不自动执行（默认）
```

### 状态更新函数注册

如果需要在每一帧执行逻辑，可以将函数注册到 `state["update"]` 中：

```python
def activate(self, state):
    def update():
        # 每帧执行的逻辑
        import math
        time = QtCore.QDateTime.currentMSecsSinceEpoch() / 1000
        state["motion"][0] = math.sin(time) * 2

    state["update"]["auto_move"] = update  # 注册到更新循环
```

注册的函数会在主循环的每一帧被调用。如需移除，可以使用 `del state["update"]["key"]`。

尽量在`update`注册循环的逻辑时，使用`plugin_id.function_name`来命名

### 完整示例：弹窗消息插件

```python
from PySide6 import QtCore, QtGui, QtWidgets
import os

PATH = os.path.dirname(__file__)

# 导入 Template（可从外部导入，或在这里定义）
class Template:
    def __init__(self):
        self._autoStart = False

    def init(self, image, mainTimer, physicsTimer, window):
        self._image = image
        self._mainTimer = mainTimer
        self._physicsTimer = physicsTimer
        self._window = window

    def activate(self, state):
        self._state = state

    def deactivate(self):
        pass

    def loadMovie(self, path: str):
        if self._image.fileName() != path:
            self._image.setFileName(path)
            self._image.jumpToFrame(0)
        self._image.start()


# 具体功能类
class Information(Template):
    def __init__(self):
        super().__init__()
        self._autoStart = False

    def activate(self, state):
        super().activate(state)
        QtWidgets.QMessageBox.information(self._window, "消息", "这是一条提示消息")


class Question(Template):
    def __init__(self):
        super().__init__()
        self._autoStart = False

    def activate(self, state):
        super().activate(state)
        QtWidgets.QMessageBox.question(self._window, "问题", "确定要执行此操作吗？")


class Warning(Template):
    def __init__(self):
        super().__init__()
        self._autoStart = False

    def activate(self, state):
        super().activate(state)
        QtWidgets.QMessageBox.warning(self._window, "警告", "这是一个警告信息")


class Error(Template):
    def __init__(self):
        super().__init__()
        self._autoStart = False

    def activate(self, state):
        super().activate(state)
        QtWidgets.QMessageBox.critical(self._window, "错误", "发生了一个错误")


# 插件元数据
pluginName = "弹窗消息"

menu = {
    "_type": "/",
    "消息": {
        "_type": "$",
        "_activate": Information()
    },
    "问题": {
        "_type": "$",
        "_activate": Question()
    },
    "警告": {
        "_type": "$",
        "_activate": Warning()
    },
    "错误": {
        "_type": "$",
        "_activate": Error()
    }
}
```

### 带 autoStart 的插件示例

```python
from PySide6 import QtCore, QtGui, QtWidgets
import random


class Reminder:
    def __init__(self):
        self._autoStart = True  # ✅ 实例属性，启动时自动执行
        self._timer = None

    def init(self, image, mainTimer, physicsTimer, window):
        self._image = image
        self._mainTimer = mainTimer
        self._physicsTimer = physicsTimer
        self._window = window

    def activate(self, state):
        print("提醒插件已加载！")

        # 启动定时器，每30秒弹窗提醒
        self._timer = QtCore.QTimer()
        self._timer.timeout.connect(self.show_reminder)
        self._timer.start(30000)
        state["reminder_timer"] = self._timer

    def deactivate(self):
        if self._timer:
            self._timer.stop()
            self._timer.deleteLater()
            self._timer = None

    def show_reminder(self):
        messages = ["该休息一下了！", "起来走动走动！", "喝口水吧！"]
        QtWidgets.QMessageBox.information(
            self._window,
            "提醒",
            random.choice(messages)
        )


class StopReminder:
    def __init__(self):
        self._autoStart = False

    def init(self, image, mainTimer, physicsTimer, window):
        self._image = image
        self._mainTimer = mainTimer
        self._physicsTimer = physicsTimer
        self._window = window

    def activate(self, state):
        if "reminder_timer" in state:
            state["reminder_timer"].stop()
            state["reminder_timer"].deleteLater()
            del state["reminder_timer"]
            print("提醒已关闭")

    def deactivate(self):
        pass


# 插件元数据（必须定义）
pluginName = "定时提醒"

menu = {
    "开启提醒": {
        "_type": "$",
        "_activate": Reminder()
    },
    "关闭提醒": {
        "_type": "$",
        "_activate": StopReminder()
    }
}
```

### 带状态更新的插件示例

```python
from PySide6 import QtCore, QtGui, QtWidgets
import math


class AutoMove:
    def __init__(self):
        self._autoStart = True  # ✅ 实例属性

    def init(self, image, mainTimer, physicsTimer, window):
        self._image = image
        self._mainTimer = mainTimer
        self._physicsTimer = physicsTimer
        self._window = window

    def activate(self, state):
        def update():
            # 每帧执行：让桌宠左右摆动
            time = QtCore.QDateTime.currentMSecsSinceEpoch() / 1000
            state["motion"][0] = math.sin(time) * 2
            state["motion"][1] = -1

        state["update"]["auto_move"] = update

    def deactivate(self):
        pass


# 插件元数据（必须定义）
pluginName = "自动运动"

menu = {
    "开启自动运动": {
        "_type": "$",
        "_activate": AutoMove()
    }
}
```
